This page will show you how to use the site




We need 2 files from you: A students file, and a projects file

Each file should be in either .csv or .tsv format (We prefer .csv since tabs are notoriously unreliable in modern text editors).


Each row in the students file represents a student.
The rows should have 6 fields each: Student number, First name, Last name, Proposer, Preferences, GPA.
Please see below for an example of what the file should look like in Excel.


Each row in the projects file represents a project.
The rows should have 3 fields each: Project title, ID of staff member who proposed it, Streams it is suitable for.
Please see below for an example of what the file should look like in Excel.
